Gestión de Alertas

Esta pantalla permite enviar una alerta usando el patrón Bridge. El sistema separa el tipo de alerta del canal de envío, para que ambos puedan cambiar por separado.

Idea del Bridge: el backend decide qué tipo de alerta crear y por qué canal enviarla sin mezclar ambas cosas en una sola clase.

El canal de correo se muestra como simulación en consola para mantener la demo simple y funcional.
