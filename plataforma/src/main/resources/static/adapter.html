Observation Adapter

Esta pantalla permite enviar datos de distintos dispositivos y convertirlos al formato interno Observation usando el adaptador del backend.

Idea del Adapter: el frontend envía datos en un formato externo y el backend los adapta a Observation, que es el formato que usa el sistema internamente.

Si quieres probar temperatura, usa el endpoint del adaptador de temperatura. Si luego agregas más rutas, esta misma pantalla puede enviar al adaptador correspondiente.
