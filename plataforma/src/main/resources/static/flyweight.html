Optimización de Medicamentos

Esta pantalla demuestra el patrón Flyweight. El sistema reutiliza la información compartida del medicamento y solo guarda en la prescripción los datos que cambian, como paciente, dosis, frecuencia y fechas.

Idea del Flyweight: el backend reutiliza un mismo objeto para cada medicamento compartido. Si usas varias veces el mismo código, la fábrica vuelve a usar la misma instancia.

Para mostrar la optimización, puedes crear dos prescripciones con el mismo código y luego revisar el cache.
